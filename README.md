# LSTM_captcha
## 基于tensorflow的LSTM网络识别验证码

### 1、前面工作
关于验证码识别，试过使用传统的machine learning方式识别，在相同样本下效果还算可以，但当迁移到别的数据集时，效果不理想。<br>
对于使用深度学习识别验证码，尝试过使用LeNet-5、AlexNet网络，可能是网络结构简单的原因，结果不收敛。故尝试用了RNN中的LSTM单元网络来识别，效果较理想。
